## Description
Mojangles is the official font used in Minecraft! This font was ported to Nintendo 3DS by [AspectX](https://github.com/3DSAspect). The Mojangles font has some weird properties which is known to cause certain issues, this font may not be fully compatible with all software. Before, using read the `README.md` on the GitHub repository.
